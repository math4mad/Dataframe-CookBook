Note

cotingencytable是理解和研究条件概率的工具

  • ref:

  • 列联表(双向表,cotingencytable,two-way table),是研究两个变量 不同水平(变量取值)之间相互关系的图表展示方法

  • 表中最大行和最大列是边缘分布

  • 表中单元格取值是两个变量的联合分布

2. two-way-table

:Gender 做为列, :Flavour 作为行, :Frequency 是值

twoway_table=unstack(data,:Gender,:Flavor,:Frequency)

3. add :Total Col

twt=combine(twoway_table,:,[:Chocolate,:Strawberry,:Vanilla]=>((a,b,c)->sum([a,b,c]))=>:Total)
