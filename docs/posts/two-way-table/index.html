生成 two-way-table

  1. 使用unstack方法生成双向表
  2. 使用combine 方法添加列
  3. 使用insert 方法添加行
include("../utils.jl")
using  RCall

Random.seed!(34343)

2. two-way-table

:Subj 做为列, :Drug 作为行, :Rate 是值

twoway_table=unstack(data,:Subj,:Drug,:Rate)

3. add :Ave Col

twt=combine(twoway_table,:,[:Pl,:Ca,:Th]=>((a,b,c)->mean([a,b,c]))=>:Ave)
